## CSS 란?
CSS(Cascading Style Sheet)

```css
selector {
    property : value;
}
```

<br>

### CSS를 적용하는 3가지 방법
* Link 태그로 css 파일을 따로 첨부
* html 문서내에서 style 태그 사용
* 요소안에 직접 입력

➡ Link 태그로 첨부하는걸 권장 함

<br>

### **꿀팁**
* 노트를 준비한다.
* 현재 상태와 만들어야 할 디자인 시안을 비교한다.
* 현 상태와 디자인 시안의 차이점을 찾아 체크리스트로 작성한다.
* 작성한 체크리스트를 하나하나 해결한다.
