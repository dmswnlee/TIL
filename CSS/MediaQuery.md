# Media Query

* Responsive web 반응형 웹을 만들기 위해서 반드시 알아야하는 css 선언

<br>

반응형 웹을 만들기 위해서는 반드시 2가지 요건이 충족되어야 함

```html
<!DOCTYPE html>
<html>
  <head>
    <meta name="viewport" content="width=device-width" />
		...
  </head>
</html>
```
```css
@media screen and (min-width:768px) {
    /* css 선언 */
}
```

* min-width
* max-width
